## Introduction

仿照ESRI公司下的Envi软件的设计方式开发的遥感影像处理系统，涵盖众多传统的遥感图像处理算法，包括图像增强以及目标分类等。
系统采用VS2010平台开发，GUI界面设计采用MFC，系统实现插件接口，用户可以在系统平台上通过实现插件的方式来满足自己的应用需求。
系统共九大模块分别为文件管理、基本工具、影像增强、影像校正、非监督分类、监督分类，变化检测，数字摄影测量，模式识别。
